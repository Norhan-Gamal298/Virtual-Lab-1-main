## ğŸ§® 2. Image Formation Model

An image is defined as:

\[
f(x, y) = i(x, y) \cdot r(x, y)
\]

Where:
- \(i(x, y)\): Illumination
- \(r(x, y)\): Reflectance

### ğŸ OpenCV (Python)

```python
import numpy as np
import matplotlib.pyplot as plt

height, width = 100, 100
image = np.zeros((height, width))

for i in range(height):
    for j in range(width):
        image[i, j] = (i + j) / (height + width)

# Display the image
plt.imshow(image, cmap='gray')
plt.title("Image Formation Model (Python)")
plt.axis('off')
plt.show()
```




### ğŸ”¬ MATLAB Code

```matlab
% Create a simple image (a gradient)
height = 100;
width = 100;
image = zeros(height, width);

for i = 1:height
    for j = 1:width
        image(i, j) = (i + j) / (height + width);
    end
end

% Display the image
imshow(image, []);
title('Image Formation Model (MATLAB)');

```

### ğŸ OpenCV (Python)
```python

PIL
(Pillow) library to work with images, and NumPy to represent them as arrays:

from PIL import Image
import numpy as np
import matplotlib.pyplot as plt

# Load an image
image_path = 'path_to_your_image.jpg'  # Replace with your image path
image = Image.open(image_path)

# Convert the image to a NumPy array
image_array = np.array(image)

# Display the image and its array representation
plt.figure(figsize=(10, 5))

# Display the image
plt.subplot(1, 2, 1)
plt.imshow(image)
plt.title("Original Image")
plt.axis('off')

# Display the array
plt.subplot(1, 2, 2)
plt.imshow(image_array)
plt.title("Image Array Representation")
plt.axis('off')

plt.show()
```

### ğŸ”¬ MATLAB Code
```matlab

% Display the image and its matrix representation
figure;

% Display the image

imshow(image);
```
